Patient Readmissions

Hospitals in US incur penalties of ~ $17 billion USD over patient Readmissions annually. However, if there are some symptoms which can predict high risk of patient readmission, the hospitals can be spared for those costs. Creating a metric for the patients is carried out in this project and also automates the process of regular patient check-ups.

Deep Learning

Deep Learning solves problems usually involving large datasets by applying deep neural networks and trained on cloud/GPUs. Deep Learning benefits from the backpropagation algorithm that can find patterns in data which cannot be done by normal machine learning approaches.

Stress Analysis using Deep Learning

While creating a 3-D printed object, thermal stresses arise that may affect the fabricated output. An efficient designing strategy would be to measure stress arising based on Laser properties being used.
By carrying out analysis of the stresses at each and every voxel, I am working on a graphical model for nearest nodes to predict these stresses. (GitHub to be updated soon!)
